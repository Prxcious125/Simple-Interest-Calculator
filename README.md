# Simple Interest Calculator

A lightweight and easy-to-use Simple Interest Calculator designed to calculate simple interest, total payable amount, and related financial values quickly and accurately.

## Features

- Calculate Simple Interest instantly
- Compute Total Amount payable
- User-friendly interface
- Fast and accurate calculations
- Beginner-friendly project structure
- Responsive and clean design

## Formula Used

The calculator uses the standard simple interest formula:

SI = (P × R × T) / 100

Where:

- **P** = Principal Amount
- **R** = Rate of Interest
- **T** = Time Period
- **SI** = Simple Interest

Total Amount:

Amount = Principal + Simple Interest
